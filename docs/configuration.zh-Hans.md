# 配置

> [English](configuration.md) · [简体中文](configuration.zh-Hans.md)

## 添加集成

**设置 → 设备与服务 → 添加集成 → Zettlab AINAS。** 会提供三种添加方式:

1. **在局域网中发现** —— 自动查找局域网内的 Zettlab 设备。
2. **输入 IP 地址** —— 发现不到时,手动填 NAS 的 IP(如 `192.168.1.50`)。继续前
   会先校验该地址确实是 Zettlab 设备。
3. **使用远程访问 ID** —— 占位;见 README 的 *已知限制*。

然后用 NAS 的网页用户名和密码**登录**(即访问 `https://<nas-ip>` 用的账号,默认
用户名 `admin`)。

## 凭据与安全

- 密码以设备公钥做 **RSA 加密**后发送,与网页端完全一致;设备返回短时令牌,集成
  会自动续期。
- 凭据存放在 Home Assistant 配置项里,除了你的 NAS 不会发往任何地方。
- ZettOS 使用自签名 TLS 证书,因此本地连接默认关闭证书校验。

## 选项

打开集成的 **⚙️ 配置** 对话框可设置:

- **轮询间隔** —— Home Assistant 刷新数据的频率。默认 `30` 秒,最小 `10` 秒。值越
  小越实时,但设备负载更高。

修改选项后集成会自动重新加载。

## 重新认证

若 NAS 密码变更,集成会请求**重新认证** —— *设置 → 设备与服务* 里会出现通知。点击
并输入新凭据即可,无需删除后重新添加设备。

## 多台设备

可以添加多台 NAS —— 每台是独立设备,以序列号为键,因此会自动防止重复添加同一台。
